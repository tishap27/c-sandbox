# C Sandbox

A collection of C programming projects and exercises developed for coursework and practice.

## Contents

This repository includes implementations of various C programming concepts:

- **Data Structures**: Linked lists (singly, circular), dynamic memory management
- **File I/O**: File handling and processing exercises
- **Algorithms**: Sorting algorithms, scheduling systems
- **Domain-Specific Programs**: Airline reservation system, flight planning, runway incursion alerts
- **Practice Code**: Pointers, structs, macros, and general C programming exercises

## Structure

The repository is organized into multiple directories containing lab assignments (`lab1`, `lab2`, etc.), weekly assignments (`WAssign1`, `Wlab1`, etc.), and standalone programs exploring specific concepts.
